# osu-api
.NET DLL library to use the osu! public api in .NET programs

## Current status
Version 1.1 done! I'm not sure it contains everything the API posts as of today (May 27 2018) though.

### Current features
* /get_user
* /get_beatmaps
* /get_scores
* /get_user_best
* /get_user_recent
* /get_match
